Expense Tracker (React)

A simple  Expense Tracker built with React. Users can add expenses, view them in a table, see the total expense, and delete expenses. Data is stored in localStorage.

Features

Add expense with name & amount

Display expenses in table format

Delete an expense

Show total expense

Persistent storage using localStorage
